# Expenseer

Expenseer is a Personal Finance Tracker application designed to offer users a platform to plan and monitor their monthly/yearly expenses and savings,
promoting financial education and assisting users to achieve their financial goals.

Full description of hte functionalities and how to use it will be provided once the MVP is ready and published.

## How to use it?

The Expenseer application will be available on public domains, once deployed the URL will be added here!
