# Expenseer

Expenseer is a Personal Finance Tracker application designed to offer users a platform to plan and monitor their monthly/yearly expenses and savings,
promoting financial education and assisting users to achieve their financial goals.

Use the application to create multiple financial budgets and expense categories.
For every expense take the time and log it in the tool, assign to the correct budget and category.

Monitor the status of you budgets in the dashboards, get even more details diving into each budget, with a history chart and a quick list of the last expenses.

For full table view of the data you have recorded, go to the expenses tab and filter your list for better visualization.

For those interested in monitoring what is going on int eh financial market, check the indexes tab, where you can search and pin stocks information and currency exchange rates.
